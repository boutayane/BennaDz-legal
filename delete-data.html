Suppression de vos données

Comment demander la suppression

Pour demander la suppression de votre compte et/ou de vos données personnelles, envoyez un e-mail à :

support@bennadzapp.com

Indiquez dans votre message :

  • l'e-mail (pour un compte vendeur) ou le numéro de téléphone (pour un client) utilisé dans l'application ;
  • si vous souhaitez supprimer tout votre compte ou seulement une partie de vos données.

Données concernées

À votre demande, nous supprimons les données personnelles vous concernant : nom, numéro de téléphone, adresse e-mail, adresses de livraison, pièce d'identité (vendeurs), photos publiées, avis et messages.

Données conservées

Certaines informations liées aux commandes peuvent être conservées de façon anonymisée le temps légalement nécessaire (comptabilité, prévention de la fraude), sans permettre de vous identifier.

Délai

Votre demande est traitée sous 30 jours. Un e-mail de confirmation vous est envoyé une fois la suppression effectuée.
